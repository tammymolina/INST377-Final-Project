This website is dedicated to Tammy's INST377 final project. It explores whether or not there is a disparity between male and female cast members in ~240 popular TV shows. The data gethered for these visualization are from the TV Maze API, which can be accessed here:
  1. https://api.tvmaze.com/shows
  2. https://api.tvmaze.com/shows/[enter show ID here]/cast
